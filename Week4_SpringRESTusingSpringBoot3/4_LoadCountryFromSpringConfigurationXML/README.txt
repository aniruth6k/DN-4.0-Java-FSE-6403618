Hands On 4 - Spring Core: Load Country from Spring Configuration XML

This module demonstrates:
- Defining a Country bean in country.xml
- Loading the bean using ClassPathXmlApplicationContext
- Logging constructor, setters, getters invocation

Key Files:
- country.xml: Spring XML configuration with country bean
- Country.java: Model class with code, name, logging in constructor, setters/getters
- SpringLearnApplication.java: Loads country bean and logs details
