Project: ObserverPatternExample

This project demonstrates the Observer Pattern in a stock market app.

Components:
- Stock (Subject interface)
- StockMarket (Concrete subject)
- Observer (Observer interface)
- MobileApp & WebApp (Concrete observers)

Features:
- Register and deregister observers
- Notify all observers on price change

By running ObserverTest.java we can see clients reacting to stock price updates in real-time.
